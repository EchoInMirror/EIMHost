# EIM

## Build

### Prerequisites

- Microsoft Visual Studio 2022
- [CMake](https://cmake.org/)

### Generate project

```bash
git clone --recursive https://github.com/EchoInMirror/EIMHost.git

cd EIMHost

mkdir build

cd build

cmake -G "Visual Studio 17 2022" -A x64 ..
```

Then open **EIMHost/build/EIMHost.sln**
